2. Dados Coletados

O sistema coleta apenas os dados necessários para a geração de planilhas de materiais:

  • Nome do cliente - Para identificação na planilha (anonimizado no histórico)
  • Data da obra - Para organização dos registros
  • Equipe - Para identificação da equipe responsável
  • Código da obra - Para rastreamento interno
  • Cidade - Para cálculos de materiais por região
  • Configurações do sistema (módulos, inversores, extras)
Nota: Não coletamos dados sensíveis como CPF, e-mail, endereço completo ou dados financeiros.

3. Finalidade do Tratamento

Os dados são utilizados exclusivamente para:

  • Geração de planilhas Excel com listas de materiais
  • Geração de romaneios em PDF
  • Estatísticas internas de uso do sistema

4. Base Legal (LGPD)

O tratamento é realizado com base em:

  • Art. 7, IX - Legítimo interesse do controlador para execução de atividades internas
  • Art. 7, V - Execução de contrato ou de procedimentos preliminares

5. Retenção de Dados

Histórico: Os dados do histórico são mantidos por 6 meses e depois automaticamente excluídos.

Arquivos gerados: Planilhas e PDFs são mantidos no servidor até exclusão manual pelo usuário.

6. Anonimização

Para proteger sua privacidade, os nomes dos clientes são anonimizados no histórico do sistema utilizando hashing, impedindo a identificação direta dos titulares.

7. Compartilhamento de Dados

Os dados não são compartilhados com terceiros. O sistema opera integralmente em ambiente controlado pelo controlador.

8. Direitos do Titular (Art. 18 LGPD)

De acordo com a LGPD, você tem direito a:

  • Confirmação da existência de tratamento
  • Acesso aos dados tratados
  • Correção de dados incompletos ou desatualizados
  • Anonimização, bloqueio ou eliminação de dados desnecessários
  • Portabilidade dos dados
  • Eliminação dos dados tratados com consentimento
  • Informação sobre compartilhamento de dados
  • Revogação do consentimento

9. Segurança dos Dados

Adotamos medidas técnicas e administrativas para proteger os dados:

  • Autenticação em endpoints de escrita
  • Validação de caminhos de arquivo
  • Restrição de acesso a dados sensíveis
  • Uso de variáveis de ambiente para credenciais
